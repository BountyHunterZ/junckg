PVC-O给水管材

外观

蓝色管材

执行标准

国际标准ISO/DIS 16422

性能

PVC-O管材具有优良的卫生性能，其工艺决定了无法添加其他物质，是安全环保的供水管材。由于管材的轴向和径向拉伸，所以管材的壁厚较薄，大大增加了通水能力。PVC-O管材的设计应力可高达28MPa和32MPa，具有突出的强度和韧度，管材易于弯曲，可根据地形进行S型铺设。

用途

产品可广泛用于自来水地下管网、饮用水输送系统、民用和工业建筑内供水输送系统、公共场所及花园、农业灌溉系统。

包装

无

安全

PVC-O管的表面光滑，施工时请勿站于管子上面。在进行粘接连接时，请保证管路内充分换气。否则会因为粘结剂中存在溶媒裂纹的残留溶媒蒸汽而容易造成PVC-O管产生小的裂纹。

说明

管件及专用粘合剂，必须在室内保管。

PVC排水管

外观

白色管材

执行标准

国家标准GB/T5836.1-2006和国际标准GB/T5836.2

性能

PVC排水管材表面硬度和抗拉强度优，管道安全系数高。抗老化性能好，正常使用寿命可达50年以上。管道对无机酸、碱、盐类耐腐蚀性能优良，除适用于民用建筑外，还适用于大部分工业污水排放及输送。管道摩阻系数小，水流顺畅，不易堵塞，养护工作量少。管材氧指数高，具有自熄性。管道膨胀系数小，为0.07mm/m,受温度影响变形量小。导热系数和弹性模量小，与铸铁排水管相比抗冻性能优良。管材、管件连接可采用粘接，施工方法简单，操作方便，安装工效高。

用途

产品可广泛用于公路、铁路路基地铁工程、废弃物填料厂、隧道、绿化带、运动场及含水量偏高引起的边坡防护等排水领域，以及农业、园业之地下灌溉、排水系统。

包装

无

安全

（1）胶粘剂、洗洁剂应随开随用，非操作人员不得动用。 （2）粘接不应有明火，通风应良好。 （3）在-10℃不宜施工，粘接点不得用火烘烤。 （4）施工人员应配备手套，粘接剂误入眼部时，立即用大量水冲洗后，送医院治疗。

说明

管件及专用粘合剂，必须在室内保管。

PVC电工套管

外观

白色管材

执行标准

标准GA305-3001和JG3050-1998

性能

PVC电工套管除具有一般PVC塑料管质轻、投资省、色泽美观等优点外，还具有抗压性能高，抗冲击性、耐热性能好，防潮耐酸碱、阻燃、绝缘、防虫等性能。

用途

产品可广泛用于室内正常环境和在高温、多尘、有震动及火灾危险的场所，也可以在潮湿的场所使用（不得在特备潮湿、有酸、碱、盐腐蚀和爆炸危险的场所使用）。

包装

无

安全

不得在特备潮湿、有酸、碱、盐腐蚀和有爆炸危险的场所使用。

说明

管件及专用粘合剂，必须在室内保管。

PE燃气管

外观

黑色管材

执行标准

标准GB15558.1

性能

PE燃气管强度高、耐环境应力开裂性能优良、抗蠕变性能好。韧性、绕性好，对基础不均和错位的适应能力强，可抵御地震和台风等恶劣环境。具有良好的耐候性和长期热稳定性。耐腐蚀、使用寿命长，时候输送天然气、液化石油气和人工煤气。内壁光滑，流通能力大，燃气流速可达20m/s。耐磨性好，抗磨损。抗低温冲击性能好，可在20-40℃温度范围内安全使用。热熔（或电熔）连接方便可靠，施工和维护方便。

用途

产品可广泛用于天然气、液化石油气、人工煤气的输配，也可用于化工、医疗、造纸等领域。

包装

无

安全

操作现场不得有明火，严禁对聚丙烯管材进行火烘弯。

PPR冷热水管

外观

白色管材

执行标准

标准GB/T17219

性能

PPR冷热水管卫生、无化学作用，安装放不安可靠，保温节能、重量轻比强度高，产品内外壁光滑、流水阻力小、输送流体系统能耗少，耐热能力高，耐腐蚀、部结构，正常使用寿命可达50年以上。

用途

（1）建筑物内的热水系统，包括集中供热系统。
（2）建筑物内的采暖系统，包括地板辐射采暖。
（3）直接饮用纯净水供应系统。
（4）中央（集中）空调供回水系统。
（5）公共场所以及花园、温室的灌溉系统。
（6）输送可排放的化学介质等工业用途。

包装

无

安全

操作现场不得有明火，严禁对聚丙烯管材进行火烘弯。

木塑产品

外观

原木质感板材

执行标准

国家标准GB/T 24137-2009 、GB/T 24508-2009 、LY/T 1613-2004,其他配套产品按相应的国家标准或甲方的企业标准验收。

性能

原木质感，色彩丰富，不易褪色、耐刮花，25年依旧焕彩如初。易清理、不费时，维护0成本。优质实木密度、韧性好，易加工。可塑性强，能非常简单的实现个性化造型，并且安装便捷。

用途

产品可广泛应用于园林景观、运输、建筑、汽车等行业，产品有地板、围栏、花箱、墙板、公园椅等，装饰美观效果佳，是一种新型的高科技环保材料。

包装

无

安全

避免产品接触腐蚀性的化学物品（特别天蓝水即乙酸异戊酯），远离高温易燃区域。。

说明

产品应该在平坦的地面上存放，并且与地面之间要设有垫板（垫板间距≤500mm,高度≥35mm)，切勿让产品两端悬空摆放，以免造成产品变形。
